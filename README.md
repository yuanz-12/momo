# 魔改墨奇码

原版拆分可见：(墨奇码表)[https://github.com/gaboolic/moqima-tables]

## 主要改动

### 字根改动

1. “弋”字根和“戊”字根归并入形近的“戈”（“戈”拆为`yd，𭠍丶`，“弋”拆为`ud，𫠠丶`）。
> 例：“式”由`yg`改为`gg`，与“诗”“试”“识”“谥”等字三码离散。

2. 增加新字根`{⿰口口}`，归并到形似的`罒`上，位于S键。
> 例：“骂”由`km`改为`sm`，与“吗”“嘛”等字三码离散。

3. 字根“囗”移动到W键上（音托，“囗”字有guó、wéi二音，也有国字框、围字框、大口框等俗名，其中“围”是最接近字理的）。
> ps：哪些字拆出“口”哪些字拆出“囗”以原版墨奇码拆分表为准，例如“西”在原版墨奇码中末形为“口”。
> 例：“圆”由`kb`改为`wb`，与“员”等字三码离散。

4. 字根“木”移到A键上（无理码）。
> 例：“嘛”由`km`改为`ka`，与“吗”等字四码离散。“术”由`md`改为`ad`，“树”由`mc`改为`ac`，与“数”等字三码离散。

5. 移除“采”字根，原来首形拆“采”的字改拆“爫”，原来末形拆“采”的字改拆“木”；“釆”字根由合并入“采”改为合并入“米”（“木”字根移到A键后，M键空旷了不少）。
> 例：“彩”由`cp`改为`vp`，与“菜”“蔡”等字三码离散。

6. 字根“斤”移动到T键上（形托，下部像大写字母T）。
> 例：“近”由`zj`改为`zt`，与“进”四码离散。

7. 移除“革”字根，原来首形拆“革”的字改拆“艹”，原来末形拆“革”的字改拆“十”。

8. 移除“刃”字根，原来首形拆“刃”的字改为拆“刀”，原来末形拆“刃”的字改为拆“丶”。
> 例：“忍”由`rx`改为`dx`，与“任”“仁”“仞”等字三码离散。

9. 将“时”的拆分改为`to，土日`（“时”本作“旹”，后世繁化成了“時”；含有部件“㞢”的字后世不少演化成了“土”或“士”，例如“志”。此处假设在一个平行世界，“旹”的正字地位没有被“時”取代，而是演化成了 ⿱土日 或 ⿱士日 ，而后按墨奇码的规则，无论“土”抑或是“士”，均位于T键）。

10. 移除“𣶒”字根，原来首形拆“𣶒”的改拆“片”，原来末形拆“𣶒”的改拆“一”。


### 拆分改动

1. “颖”“颍”等声旁“顷”的字改为取包围结构。
> 例：“颖”由`by`改为`bh`，拆分为⿹顷禾，顷继续拆分为匕页。最终结果为：匕页禾。

2. “渊”字由`dl，氵丿米丨`改为`dm，氵⿰丿丨米`，与“粛”等字成系统。

3. 移除“止”的三种拆分，改为`bl，卜丨`和`lu，丨上`两种拆分。


### 兼容码改动

1. 为“曲”“典”二字增加兼容码，首形拆“艹”。

2. 为“未”“末”二字增加兼容码，首形拆“二”。

3. 为“叔”“尗”二字增加兼容码，首形拆“卜”。

4. 移除“剑”的兼容码`rd，人刀`。

5. 移除“丰”的兼容码`af，一𰀁`，保留兼容码`al，一丨`。

6. 移除“瓤”的兼容码`xg，襄瓜`。

7. 移除“极”的兼容码`mn，木乀`。

8. 移除“伊”的兼容码`rp，亻丿`。

9. 将“甩”的兼容码`kv，冂乚`扶正，保留第一兼容码`ov，月乚`的位置，将原墨奇码正拆`yv，用乚`降为第二位兼容码。
